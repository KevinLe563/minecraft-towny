# minecraft-towney

Plugin Configs for a Towney MC server with the boys

## Rules

1. No PVP unless consented by both players.
2. No griefing within the town. Outside is fair game.
3. Glitches, bugs and unbalanced / unintentional mechanics should be reported immediately. Abusers will be banned.
